# Digital Marketing

> Projeto sistema de mailmarketing com agendamento de disparos de email, utilizando api mailgun.
> Projeto criado com o intuito de testar conhecimentos de React.

Tecnologias utilizadas no Back-end:

- Mongodb
- Mongoose
- Passport JWT
